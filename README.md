# Kubernetes Research & Demo Projects

A collection of 10 Kubernetes-focused research/demo repositories covering ingress, autoscaling, failover, logging, blue/green deployment, resource tuning, and multi-service application patterns.

## Repositories

1. **k8s-smart-ingress-router**  
   Adaptive ingress routing demo with simulated backend health scoring and routing decision rendering.  
   Repo: <https://github.com/biprajit007/k8s-smart-ingress-router>

2. **k8s-multi-language-microservices**  
   Multi-service Kubernetes example combining Python, Go, and Node.js components.  
   Repo: <https://github.com/biprajit007/k8s-multi-language-microservices>

3. **k8s-auto-healing-system**  
   Auto-healing research demo that analyzes workload failure events and suggests remediation.  
   Repo: <https://github.com/biprajit007/k8s-auto-healing-system>

4. **k8s-real-time-log-analyzer**  
   Log analysis and lightweight dashboard demo for Kubernetes-style observability workflows.  
   Repo: <https://github.com/biprajit007/k8s-real-time-log-analyzer>

5. **k8s-hpa-custom-metrics**  
   Horizontal Pod Autoscaler demo using custom metrics and load simulation.  
   Repo: <https://github.com/biprajit007/k8s-hpa-custom-metrics>

6. **k8s-onprem-load-balancer**  
   On-prem Kubernetes load balancing lab with MetalLB and NGINX examples.  
   Repo: <https://github.com/biprajit007/k8s-onprem-load-balancer>

7. **k8s-stream-processing-platform**  
   Stream processing lab with producer/consumer components for Kubernetes-oriented experiments.  
   Repo: <https://github.com/biprajit007/k8s-stream-processing-platform>

8. **k8s-blue-green-deployment-system**  
   Blue/green deployment demo focused on release switching and traffic cutover patterns.  
   Repo: <https://github.com/biprajit007/k8s-blue-green-deployment-system>

9. **k8s-service-failover-system**  
   Primary/backup service failover demo using health-based active target selection.  
   Repo: <https://github.com/biprajit007/k8s-service-failover-system>

10. **k8s-resource-optimizer**  
    Resource recommendation demo for CPU and memory request/limit tuning.  
    Repo: <https://github.com/biprajit007/k8s-resource-optimizer>

## Coverage Areas

- Ingress and routing
- Microservices architecture
- Auto-healing and remediation
- Logging and observability
- Autoscaling with custom metrics
- On-prem load balancing
- Stream processing
- Blue/green deployments
- Service failover
- Resource optimization
